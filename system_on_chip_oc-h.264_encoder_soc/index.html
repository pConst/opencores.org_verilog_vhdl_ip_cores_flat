Description

The goal is to develop an H.264 Encoder SoC (System-on-Chip) solution based only on open-source blocks.

Status

2009-08-24:
A software-team is assembled and are now looking into the first task, which is to get profiling information about the VLC x264 encoder.
We basically need this to understand what functions are suitable to extract from the VLC-SW and then be implement as HW instead. We also need to discuss the pros/cons of moving different functions to hardware (size/performance issues etc).
