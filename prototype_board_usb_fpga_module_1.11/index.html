Description

The ZTEX USB-FPGA-Module 1.11 is a Spartan 6 FPGA board with USB 2.0 interface, 64 MB DDR-SDRAM and Flash memory.

Features

  • High-Speed (480 MBit/s) USB interface via Mini-USB connector (B-type)
  • Cypress CY7C68013A EZ-USB-Microcontroller
  • Xilinx Spartan 6 XC6SLX9, XC6SLX16 or XC6SLX25 FPGA
  • 90 General Purpose I/O's (GPIO):
    • 82 FPGA GPIO's
    • 8 EZ-USB FX2 GPIO's
  • 15 special I/O's (SIO):
    • 4 pins for FPGA configuration via JTAG
    • 2 pins for 1 serial ports
    • 2 pins I 2 C interface (2 pins)
    • 2 interrupt pins
    • 3 timers
    • 1 Wakeup pin
    • 1 Breakpoint pin
  • 64 MByte DDR SDRAM:
    • 200 MHz clock frequency
    • 16 Bit bus width
    • Up to 800 MByte/s data rate
    • Easy to use hard memory controller included Spartan 6 FPGA with 6 ports. (examples included in SDK)
  • MicroSD socket for standard and high capacity (SDHC) microSD cards (the cards must support the SPI mode)
  • 128 Kbit EEPROM memory (can be used to to store the firmware)
  • FPGA configuration via USB (no JTAG adapters or other additional utilities required) or from Flash memeory
