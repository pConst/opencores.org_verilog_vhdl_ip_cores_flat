Description

This is an OPB-compatible VGA character display for the Spartan 3E development board, which does not contain DACs.
The core is very small, requiring only 3 BRAMs and 533 slices.
All access is through write character commands, similar to an LCD display. 3-bit color is supported, as well as inverted characters.
Current resolution is 640x480, 75x55 characters

Features

- Ease of use
- Includes write string function
- Individual characters can also be written
- Small footprint
