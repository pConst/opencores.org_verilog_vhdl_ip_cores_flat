[EN] Description


The aim of the project is to develop oscilloscope functions (conversion control, trigger, FFT, ...) in several cores and implement a usefull oscilloscope in a development board. The results are shown in a computer with a simple software. The software will be developed using LGPL tools and distributed under such license, like the cores.
The beginig of this project is part of a universitary work in National University of San Luis, Argentina. For now, it is only a unipersonal project, but in some months it will have more maintainers.

[EN] Features


- PC interconnection
· EPP
- Configurable acquisition
- Multiple inputs
- Several trigger options
- Software
· GUI for windows and linux
· Qt library

[ES] Descripción

La primera parte de este proyecto forma parte de un trabajo universitario en la Universidad Nacional de San Luis de Argentina. Por esta razón, por ahora, solo se mantendrá en forma unipersonal.
El objetivo del proyecto es desarrollar funciones de osciloscopios (control de conversión, trigger, FFT, ...) en varios módulos e implementar un osciloscopio funcional en una placa de desarrollo. Los resultados se mostrarán en un programa sencillo de computadora.
El programa y los IP Cores serán desarrollados bajo licencia LGPL.
