Definition

A Complex arithmetic library for arithmetic operations is needed in many signal processing applications. This project will present a complex operations library for SystemC based designs. Some of the operations like multiplication, division and square root are based on Cordic algorithms in order to reduce the resources needed for implementation. Eventhough the library is based on the complex library of the Agility Compiler Software but nearly all of the operations were modified and improved. The operations included within this project are;

  • addition (done)
  • subtraction (done)
  • multiplication (done)
  • division (done)
  • polar to cartesian representation conversion (done)
  • cartesian to polar representation conversion (done)
  • conjugate (done)
  • square root (done)
