Description

PDP-1 reimplementation using an FPGA. The goal is to run old software like Spacewar!, the music compiler, and Expensive Typewriter on current FPGA hardware. Initial platform is a Spartan 3A starter kit.
For more information on the original machine, see http://pdp-1.computerhistory.org/pdp-1/
