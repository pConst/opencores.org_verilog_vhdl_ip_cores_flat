Description

Converts Raspberry Pi into a JTAG programmer (STAPL protocol). Supports two JTAG chains through 26-pin RPi GPIO P1 connector. The TCK rate is ~1MHz. Works reliably for distance up to 1m, with LVDS level converter tested successfully for distance up to 15m.
Ported from Actel STAPL Player v1.1, which is based on JAM STAPL Player v2.2.
