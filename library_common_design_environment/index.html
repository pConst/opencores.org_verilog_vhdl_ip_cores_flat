Description

Common Design Environment (CDE) is a library of modules that usually require replacement with specific hard
macros when the design is retargeted to a IC process. By using modules from the CDE library it will be possible
to easily make this substitution with an ip-Xact enabled tool flow without having to touch the original rtl code.
