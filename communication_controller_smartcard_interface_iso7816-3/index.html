Description

This project's aim is to provide the smart-card side of an ISO 7816-3 interface.

Features

- Detects reset and sends ATR (Answer to Reset).
- Translates between the raw ISO7816-3 serial data and a Wishbone compliant format.
- Provides examples modules that interpret commands sent over the interface.

Status

- A working prototype is ready.
- No work is currently being done.
- Current version available for download from http://www.opencores.org/pdownloads.cgi/list/iso7816-3 (the download tab).
