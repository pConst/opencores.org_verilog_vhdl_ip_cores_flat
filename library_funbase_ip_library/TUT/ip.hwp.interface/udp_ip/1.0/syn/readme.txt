These .qsf files are pinmaps that connect the logical names in VHDL
(e.g. RESETn) to FPGA's physical pins (e.g. PIN_V2).

qsf = Quartus settings file

You can import these QSF files to Quartus when using the following boards:


Altera DE2 with Davicom DM9000A
Altera Stratix II S180 DSP development board with SMSC LAN91C111
