--------------------------------------------
BASIC TEST for hibi_v3 using two r4 wrappers
--------------------------------------------

To run the simulation with Modelsim:

   1. Change to this directory in Modelsim and run 
      'do compile_all.do' in transcript window

   2. Follow the instructions printed in transcript window

   3. Wave window should look like 'expected_result.png'

Other simulators:

   1. Check the files needed from basic_test_hibiv3.do file

   2. Compile all needed files

   3. Add signals found in add_signals.do to simulation

   4. Run simulation for 350 ns
