Description

Open-source implementation of a versatile UDP/IP core for FPGAs.
C/C++ Software library for configuring the core and transmitting standard C types like characters, integers, floats and doubles.
Hardware interface for transmitting standard C types like characters, integers, floats and doubles.
The UDP/IP core can transmit and receive data to and from any PC. The only requirement is the use of the configuration function provided by the software library in order to initialize an internal lookup table.
