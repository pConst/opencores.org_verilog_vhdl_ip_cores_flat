An inexpensive PCI FPGA development board

This is a port of the Opencores PCI core ported to the Enterpoint Raggedstone1 PCI card.
This is a very inexpensive card: (~$100 USB). You can order one from Enterpoint:
http://enterpoint.co.uk/moelbryn/raggedstone1.html

A PCI based FPGA card

-Inexpensive
-PCI core works
-All the code is in CVS
-Uses the OC PCI core

Version is in cvs

To check this project out of cvs:
cvs -d:pserver:anonymous@cvs.opencores.org:/cvsroot/anonymous checkout raggedstone
