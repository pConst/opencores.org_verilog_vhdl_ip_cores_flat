Wishbone Monitor Controller Accelerated VGA Core

Description

Wishbone Monitor Controller Accelerated VGA Core adds a acceleration and palette support to the VGA core module. All it's interfaces are WishboneTK compatible and thus can be used to integrate with into a SoC design. It has no directly mapped pixel memory seen from the CPU side. It is intended as a DEMO application rather than a real-world example.
