Description

GroundHog 2009 is a benchmark suite for reconfigurable architectures in the mobile domain. The benchmark suite can be downloaded from http://cc.doc.ic.ac.uk/projects/GROUNDHOG/ .
The benchmark suite includes designs described in a high-level format that is non-synthesizable. For this reason, this project on OpenCores is a space where individuals in the community can release synthesizable (and compilable) versions of their implementations of the 6 benchmarks.
The benchmarks are:
GH09.B.1 - Port expander and keypad controller
GH09.B.2 - Glue Logic
GH09.B.3 - AES encryption cypher
GH09.B.4 - Data compression using Lempel-Ziv
GH09.B.5 - Bridge chip
GH09.B.6 - 2D convolution
Hardware test bench and example RTL for the design cases can be found at
http://www.cs.tut.fi/projects/GroundHog2009/index.html .
To view synthesizable designs check the download section.
Also, to contribute your designs to the repository please contact us.
